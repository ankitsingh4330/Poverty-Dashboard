The Poverty Analysis Dashboard is a data visualization project created using Microsoft Power BI.
The dashboard analyzes poverty-related data to provide insights into population distribution, poverty levels, and socio-economic indicators.

The purpose of this project is to transform raw data into interactive visual insights that help understand poverty trends and patterns.

🎯 Objectives

Analyze poverty statistics using visual analytics

Identify patterns and trends in poverty levels

Present data in a clear and interactive dashboard

Support data-driven decision making

🛠 Tools & Technologies Used
Tool	Purpose
Power BI	Dashboard development and visualization
Excel	Data source
Power Query	Data cleaning and transformation
DAX (Data Analysis Expressions)	Creating calculated measures and KPIs

📈 Key Insights

The dashboard helps to:

Identify regions with high poverty levels

Compare population distribution and poverty

Understand economic inequality patterns

Monitor poverty trends over time
